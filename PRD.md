# 인간주식 (가칭) — PRD v5

> 친구·가족·크루가 **소규모 폐쇄 그룹**을 이뤄 서로의 "주식"을 거래하는 온체인 소셜 게임.  
> 검증된 일과가 진짜 가치를 만들고, **AI가 속보로 중계**하고, **친구들이 "내일의 너"에 베팅**한다.  
> 각 크루 = 독립 미니 거래소. 핵심 원칙: 오픈형 금지(변질방지).

---

## 1. 제품 한 줄 설명

친구·가족·크루가 소규모 폐쇄 그룹을 만들어, 각자의 일과와 성취를 “주식 가치”로 바꾸고 서로의 성장 가능성에 베팅하는 소셜 게임이다.

---

## 2. 핵심 모델

```txt
[사진 인증 = 실적] → [AI 심판이 적정가(FV) 공시] → [크루 내 본딩커브 시장에서 시장가 형성]
 ↑ 크루원 매수·매도 + 내일 예측 베팅
[오늘의 TMI = 뉴스] ──── AI 속보 연출 → 단기 변동 ───────┘
```

- 단위 = 크루(폐쇄 그룹). 종목·시장·토큰·랭킹·베팅 전부 크루 안에서만 작동한다.
- AI = 심판(오라클). 적정가만 공시하고 직접 매매하지 않는다.
- 추세는 AI 실적이 만들고, 출렁임은 크루원 심리·베팅이 만든다.
- 시즌제로 운영하며 시즌 종료 시 전원 리셋한다.

## 3. 잠근 설계 원칙

| 원칙 | 결정 |
| --- | --- |
| 운영 형태 | 폐쇄 소규모 크루 단위. 오픈형 금지. 변질 방지 목적 |
| 시장 범위 | 크루 = 독립 미니 거래소. 같은 사람도 크루마다 주가가 다를 수 있음 |
| 초기가 | 크루 내 전원 동일 고정 단가로 시작 |
| 시드/경제 | 전원 10,000,000 HSC 가상 HSC 지급. 시즌당 1회. 리필 없음. 통합 지갑 사용 |
| 난이도·경중 | 두 축 분리. 난이도 = AI 기준선 + 개인 상대 기준 + 크루 보정. 경중 = 목표 정렬도 |
| 베팅/숏 | 네이키드 공매도 금지. 대신 크루 내 “내일” 예측 베팅 롱/숏 허용 |
| 베팅 정산 | 풀 분배 방식. 동점은 중립밴드 처리 후 전액 환불. 1건/1일 상한 적용 |
| 배당 | 홀더 배당은 post-MVP로 보류 |
| 상장폐지 | 무활동 3-strike → 폐지. 해당 종목 보유분 전액 소각. 경고 시 매매 동결. 자발 halt는 면제 |
| AI 역할 | AI는 심판/오라클만 담당. 직접 매매 없음. 결과는 속보체로 연출하되 숫자는 실제 계산값만 사용 |
| FV vs TMI | 성취는 펀더멘털로 크게 반영. TMI·속보는 뉴스처럼 작게, 시장가에만 영향 |
| 미인증 | 폭락이 아니라 점진 감쇠. 강제 동결은 복구 쉽게. 자발 halt는 무페널티 |
| 시장 형식 | 본딩커브 AMM + 소수 인원 튜닝 |
| 시즌 | 시즌제 운영. 종료 시 전원 시드·포지션 리셋 |
| 성장 방식 | 친구 그래프 기반 바이럴. BeReal·Locket처럼 크루 단위 확산 |

## 5. 기능 모듈

### A. 크루 & 종목 생성

- 크루 생성·초대·멤버 관리: 폐쇄형, 초대 코드/링크, 정원 3~30명.
- 온보딩 & 종목 등록: 크루 가입 = 상장. 전면 옵트인. 목표 영역 선언.
- 상장 & 초기가: 크루 내 전원 동일 고정 단가. 본인 초기 보유 수량만 선택.
- 크루 토큰 이코노미: 전원 10,000,000 HSC 가상 HSC, 시즌당 1회, 리필 없음.
- 통합 지갑: 주식 매수 + 예측 베팅을 하나의 지갑에서 사용.
- 거래 수수료: 크루 풀 + 종목 본인 로열티.

### B. 일과 입력 & 가치 산정

- 성취 인증: 목표 선언 → 실시간 사진 → 위조 방지.
- 난이도·경중 산정: 태스크 가중치 = 난이도 × 경중.
- 난이도: AI 카테고리 기준선 + 개인 상대 기준 + 크루 보정.
- 경중: 온보딩 목표와의 정렬도.
- MVP: AI prior + 자기신고 + 크루 이의 버튼.
- 오늘의 TMI: 자유 일상 입력 → 호재/악재/중립 해석 → 지연 효과. FV 미반영, 시장가만 흔듦.
- AI 심판: 비전 판정, 확신도 낮으면 조정 보류, 하루 1회 종가 결산.

### C. FV 엔진

```txt
FV_t = FV_t-1 × (1 + r_t)
r_t = 성취 리턴 + streak 보너스 + 마일스톤 점프 - 미인증 감쇠
```

기본값:

```txt
FV0 = 100
k = 0.05
streak = 0.4%/일, cap 3%
milestone = +5%, +12%, +25%
미인증 감쇠 = -2.5% ~ -5%
일일 캡 = -8% ~ +12%
하한 = 30
```

### D. 거래 시장

- 본딩커브 AMM: 매수=민팅, 매도=번.
- FV 연동: FV가 커브 기준선.
- 소수 인원 튜닝: 완만한 커브, 1인 보유·거래 상한.
- 안정장치: 일일 변동폭 상한, 서킷브레이커, 급변 쿨다운, 담합·펌핑 탐지.

### E. 보상 & 디스커버리

- 응원 경제: 종목 보유 = 그 사람 성장에 베팅.
- 크루 리더보드: 주간 MVP, 최고 상승 종목, 최고 수익 투자자.
- 대시보드: FV vs 시장가 차트, AI 판정 근거, 포트폴리오, 크루 보드.

### F. 소셜 콘텐츠 & 예측 베팅

- AI 속보 중계: 금융 속보·분석 로그체로 연출.
- 크루 배너 알림.
- “내일의 너” 예측시장: Long/Short.
- pari-mutuel 풀 분배.
- 동점(push): 중립 밴드 안이면 전액 환불.
- 1건/1일 베팅 상한.

### G. 라이프사이클·거버넌스·안전

- 3-strike 상장폐지.
- 경고 = 사이드카, 매매 동결.
- 활동 재개 시 동결 해제.
- 자발 halt는 무페널티.
- 탈퇴/잊힐 권리.
- 사진·TMI는 크루 내 한정 공개.
- 테스트넷·가상 토큰 전용.
- 얼굴·위치 블러, 분쟁 재심.

## 6. 주요 화면

1. 온보딩: 닉네임, 목표 영역, 종목명, 초기 보유 수량, 크루 가입/생성.
2. 크루 홈: 크루 이름, 시즌 기간, 지갑, 종목 리스트, 상승률 TOP, AI 속보, 예측 카드.
3. 종목 상세: 시장가, FV, 차이, 최근 실적, AI 근거, 매수/매도, 예측 베팅.
4. 일과 인증: 목표, 사진 촬영, 메모, 자기신고 난이도/중요도, AI 판정.
5. AI 속보 피드: 호재/악재/중립/TMI/공시/지연 효과.
6. 리더보드: 주간 MVP, 최고 상승 종목, 최고 수익률 투자자, 예측 적중률, streak.

## 9. 디자인 방향

```txt
토스 + 증권앱 + 게임 리더보드 + 친구 장난
```

- 진지한 금융앱처럼 보이지만 실제로는 친구끼리 노는 성장 게임.
- 너무 무겁지 않게.
- 친구가 상장된 느낌.
- 속보는 웃기고 과장되게.
- 숫자와 차트는 보기 쉽게.

핵심 UI 문구:

- 오늘의 내 주가
- 친구들이 보는 내일의 나
- AI 공시
- 내 주주
- 내가 산 친구들
- 오늘의 호재/악재
